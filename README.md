# CMSS Group Project

Simulation code for the Computational Models of Social Systems assignment.

The project compares clustered seeding and high-degree seeding in a Watts–Strogatz threshold model. We vary adoption thresholds and rewiring probabilities to see when clustered seeding leads to higher adoption.

The repository contains the simulation code, generated result tables and final plots.

## Run

```bash
python -m src.run_final_experiment
```
This reruns the full experiment and saves updated CSV files in data/ and plots in plots/final/.
